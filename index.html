DeepXTools

Various deep learning tools.
Free open-source software under GPL-3 license.

Requirements

Windows 10
NVIDIA Card 4GB+

Installation

Download install-DeepXTools-NVIDIA.bat

Place it to the root of any disk. For example C:\
Run file.
After installation finishes you will see C:\DeepXTools directory.

Mask Editor



Mask annotation tool.

Allows to edit and view the mask for images.

Deep Roto



Mask training tool.

You annotate only a few images. Train the neural network. Get a predicted mask for all images.
